      概述

      使用 Prometheus 和 Grafana 对Linux服务器进行监控。

      Prometheus 是什么

      Prometheus 是一个开源的监控软件，通过HTTP的方式从远程机器(通过 node_exporter)收集数据并存储在服务器上。

      Grafana 是什么

      Grafana 是一个功能强大的可视化工具，支持多种数据源。
