Go 笔记（一）

Go 常用命令

  • go get：获取远程包（需安装git or hg）
  • go run：运行程序
  • go build：测试编译，检查是否有编译错误
  • go fmt：格式化代码
  • go install：编译包文件并编译整个文件
  • go test：运行测试文件（ 以 _test.go 结尾）
  • go doc：查看文档(整个包的文档)
  • godoc fmt Println:查看fmt包中Println函数的文档
