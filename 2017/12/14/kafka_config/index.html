Kafka 主要配置参数概览

Veröffentlicht am 2017-12-14

Broker

Server Basics

键 设定值 描述
broker.id none 每个broker的唯一标识

Socket Server Settings

键 设定值 描述
host.name ip broker的主机名
num.network.threads 3 处理网络请求的线程数
num.io.threads 8 处理磁盘的线程数
socket.send.buffer.bytes 102400 满这个数才发送
socket.receive.buffer.bytes 102400 满这个数才写入磁盘
socket.request.max.bytes 104857600 单次最大请求字节数(防止OOM)

Log Basics

键 设定值 描述
log.dirs 日志存放路径
num.partitions 1 每个topic默认 partition 数

Log Retention Policy

键 设定值 描述
log.retention.hours none 保存多长时间日志
log.retention.bytes 1073741824 超过这个大小，日志被裁减（不知怎么用）
log.segment.bytes 1073741824 累计日志大小超过这个，新建一个

Producer

Producer Basics

键 设定值 描述
metadata.broker.list broker列表，host:port
partitioner.class 分区策略
producer.type sync 日志发送方式
compression.code none 压缩格式
serializer.class DefaultEncoder

Consumer

键 设定值 描述
group.id 设置group

同一topic的一条消息只能被同一个consumer group内的一个consumer消费，但多个consumer group可同时消费这一消息
